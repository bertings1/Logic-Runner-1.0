# Logic Runner - Propositional Logic Game

An interactive educational game for learning propositional logic concepts.

## Features
- Modern UI inspired by OpenAI
- Drag-and-drop matching game
- Truth table challenges
- Logic puzzles
- Progress tracking

## How to Run
1. Download all files
2. Open `index.html` in your web browser
3. No installation required!

## Game Instructions
1. Drag operators to match with their symbols
2. Complete truth tables
3. Solve logic puzzles
4. Progress through 4 levels

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts (Inter)
