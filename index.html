Tutorial

How to Play

  1. Choose a difficulty & mode then press Start.
  2. Each question is unique; questions are randomized and won't repeat across difficulties.
  3. In Timed mode the floating timer starts only when you press Start.
  4. Use Retry to restart the current run or Quit to return to settings.
