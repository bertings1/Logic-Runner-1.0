Propositional Logic — Quick idea

Propositional logic deals with statements that are either true or false. Practice using connectives like AND, OR, NOT and IMPLIES through short interactive challenges.

Tutorial
Connectors & Semantics

AND (∧) — true only when both p and q are true.
OR (∨) — true when at least one of p or q is true.
NOT (¬) — inverts truth.
IMPLIES (→) — false only when p is true and q is false.

How to Play

  1. Choose difficulty and mode, then press Start.
  2. Question types: symbol match, sentence fill, truth-table tasks (drag / mcq / type / dropdown / partial).
  3. Correct answers increase score. Wrong answers lose a life. Run ends at 0 lives.
  4. In Timed mode, each question has a countdown — answer before time runs out.
About
Learning Goals

Logic Runner is a practice tool to build fluency in propositional logic: translating sentences into logical operators, reading truth tables, and applying logical reasoning under mild time pressure.

Designed for students. Use it often and the patterns will stick.
