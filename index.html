Propositional Logic — Quick idea

Propositional logic is about statements that can be true or false. You'll build and test expressions using operators like AND, OR, and NOT — practice by playing the levels.

Tutorial
Connectors & Semantics

AND (∧) is true when both parts are true. OR (∨) true when at least one is true. NOT (¬) inverts truth. Practice by building sentences and checking truth outcomes in the game.

How to Play

  1. Choose difficulty and mode, then press Start.
  2. Each level contains unique questions (randomized every run).
  3. In Timed mode the floating timer starts per question.

Examples

Sentence Builder: "A ___ B (true if both are true)" — drop "AND".

Truth Table: Fill T/F for each row in inclusive/exclusive/negation tables.
