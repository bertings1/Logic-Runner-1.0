Tutorial

How to Play

  1. Choose difficulty and mode. Each run uses unique logic tasks.
  2. Symbol Match: Drag logic name to the correct symbol (AND, OR, IMPLIES, NOT).
  3. Sentence Builder: Fill in the blank with the correct logic connector.
  4. Truth Table Task: Fill T/F results for tables such as AND, OR, XOR, Negation.
  5. Lives/Timer: Score more by getting answers right before your lives/timer runs out.

More features: mobile-friendly, leaderboard, player name, sounds, modal feedback. Try it now!
