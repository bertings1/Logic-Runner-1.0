Propositional Logic — Quick idea

Propositional logic is about statements that can be true or false. Build and test expressions using operators like AND, OR, NOT, IMPLIES, XOR — practice by playing the levels.

Tutorial
Connectors & Semantics

AND (∧): true only when both p and q are true.
OR (∨): true when at least one of p or q is true.
NOT (¬): inverts truth.
IMPLIES (→): false only when p is true and q is false.
XOR (⊕): true when exactly one of p or q is true.

How to Play

  1. Choose difficulty and mode, then press Start.
  2. Questions appear in mixed-mode: symbol matching, sentence gap-fill, or truth-table tasks (drag/MCQ/type/dropdown/partial).
  3. Score points for correct answers; wrong answers cost a life. Lose all lives and the run ends.
  4. In Timed mode, each question has a countdown — answer before time runs out.
  5. Use Retry during a run to quickly restart, or Quit to return to the settings panel.
About
Learning Goals

Logic Runner is a short, interactive practice tool to build fluency in propositional logic: translating sentences to operators, recognizing truth tables, and applying logical reasoning under time pressure.

Designed for BSIT students — Group 3 Block 2N. Use it to practice truth tables, connectives, and basic reasoning skills.
