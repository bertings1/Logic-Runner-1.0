What is Propositional Logic?

Propositional logic studies statements that are either true or false. Using connectives such as AND, OR, NOT, and IMPLIES, we analyze how complex expressions behave. Logic Runner turns these ideas into short, fun challenges designed to strengthen your reasoning skills.

Tutorial
Logical Connectives

AND (∧): True only if both p and q are true.
OR (∨): True if at least one of p or q is true.
NOT (¬): Inverts the truth value.
IMPLIES (→): False only if p is true and q is false.

How to Play

  1. Select a difficulty level and game mode.
  2. Solve tasks like symbol matching, sentence translation, and truth-table filling.
  3. Correct answers increase your score; wrong answers cost one life.
  4. In Timed mode, answer before the countdown ends.
About Logic Runner
What You’ll Learn

Logic Runner is designed to build skill and confidence in propositional logic. You’ll learn to:

  • Translate English statements into logical expressions.
  • Interpret and complete truth tables.
  • Recognize common logical patterns and structures.
  • Evaluate compound expressions step-by-step.
  • Apply reasoning quickly under mild time pressure.
Example Problems
  • Translate: “If it rains, the ground is wet.” → p → q
  • Fill truth table: p = T, q = F → p ∧ q = F
  • Evaluate: ¬(p ∨ q) when p = F, q = F → True
  • Match: “not p” → ¬p
