Quick Tips
  1. Drag operator names to symbols or fill in the correct answer!
  2. Timed mode: solve before the timer runs out.
  3. Use Retry or Quit to restart or exit at any time.
  4. Points = logic & table accuracy.
Tutorial
  1. Choose difficulty and mode
  2. Each question: follow the prompt and drag/fill the logic.
  3. Score points for correct moves/thinking. Lose lives if wrong.
  4. Timed = more pressure, more score.
