Propositional Logic — Quick idea

Propositional logic is about statements that can be true or false. Build and test expressions using operators like AND, OR, NOT, IMPLIES, XOR — practice by playing the levels.

Connectors & Semantics

AND (∧) true when both are true. OR (∨) true when at least one is true. NOT (¬) inverts truth. IMPLIES (→) false only if true→false. XOR (⊕) true if exactly one is true.

How to Play

  1. Choose difficulty and mode, then press Start.
  2. Each level contains randomized questions: match, sentence, truth-table (various answer modes).
  3. In Timed mode the floating timer counts down.
