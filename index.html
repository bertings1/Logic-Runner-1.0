Tutorial
Connectives & How to Play

These are the four basic operators in propositional logic:

  • AND (∧) — true only if both p and q are true
  • OR (∨) — true if at least one of p or q is true
  • NOT (¬) — flips the truth value
  • IMPLIES (→) — only false when p is true and q is false

Gameplay Tips

  1. Select your difficulty and mode.
  2. Question types include:
    • Operator matching
    • Symbol translation
    • Truth table completion
    • Expression evaluation
  3. Correct answers increase score; mistakes cost lives.
  4. Timed Mode requires answering before the countdown ends.

Example Tasks

  • "not p" → ¬p
  • p=T, q=F → p → q = False
  • "If the switch is on, the light turns on." → p → q
  • Evaluate ¬(p ∧ q) when p=T, q=F → True
